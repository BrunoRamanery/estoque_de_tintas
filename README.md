# Sistema de Estoque de Tintas

Sistema web desenvolvido em PHP e MySQL para controle de estoque de tintas e impressoras.

## Funcionalidades
- Cadastro de tintas
- Cadastro de impressoras
- Controle de quantidade
- Controle por cor, mês e ano
- Login de usuário
- Validação com CSRF

## Tecnologias
- PHP
- MySQL
- HTML
- CSS
- XAMPP

## Como executar
1. Coloque a pasta do projeto dentro do `htdocs`
2. Inicie Apache e MySQL no XAMPP
3. Importe o banco de dados
4. Configure a conexão com o banco
5. Acesse pelo navegador

## Estrutura
- `app/`
- `config/`
- `css/`
- `funcoes/`
- `impressora/`
- `includes/`
- `repositorio/`
- `servicos/`
- `usuario/`
